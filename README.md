# stanzme-website-redesign
Building a production website for a medical assistive technology company serving children with special needs using Squarespace and custom HTML/Javascript/CSS enhancements

## Overview
Redesign and enhancement of the Stanzme website for a real client specializing in assistive products for individuals with special needs.

- Enhance accessibility

- Improve SEO and performance
